https://github.com/jtesta/ssh-audit
Es una herramienta que nos ayuda a recopilar informacion sobre las configuraciones y tenconologias que usa un servicio de ssh
# Instalacion
```shell-session
git clone https://github.com/jtesta/ssh-audit.git && cd ssh-audit
```

# Uso
```shell-session
./ssh-audit.py 10.129.14.132
```
