# Descipcion
La configuración predeterminada del atributo ms-DS-MachineAccountQuota de Active Directory permite a todos los usuarios del dominio agregar hasta 10 cuentas de máquina a un dominio. Powermad incluye un conjunto de funciones para explotar ms-DS-MachineAccountQuota sin adjuntar un sistema real a AD.
